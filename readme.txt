The MainWP Key Maker plugin copies settings for the MainWP Bulk Settings Manager Extension. 


== Description ==

**Note: This plugin is only useful if you are using [MainWP](https://wordpress.org/plugins/mainwp/) and the [Bulk Settings Manager Extension](http://mainwp.com/extension/bulk-settings-manager/).**

Install the Key Maker plugin on any WordPress site to quickly copy form settings for import into MainWP Bulk Settings Manager Extension.

== Installation ==

1. Upload the MainWP Key Maker folder to the /wp-content/plugins/ directory
2. Activate the MainWP Key Maker plugin through the 'Plugins' menu in WordPress
